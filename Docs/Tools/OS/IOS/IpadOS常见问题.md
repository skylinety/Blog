## Apple pencil 断触

### 软件问题

```sh
使用系统自带的备忘录尝试是否为软件问题
```

部分软件可能手触屏幕会有断触问题

### 还原设置

```sh
关掉蓝牙重新连接 Pencil

重启 iPad 后再次连接 Pencil

打开 ipad 设置—>通用—>还原—>还原所有设置
```

上述操作只是还原设置，不会删除数据，但是人脸识别和密码，壁纸会还原

### 硬件检测

```sh
去除贴膜，笔尖套

更换笔头尝试

换个 ipad 试下是否为笔的问题

```

这些操作都不行，联系售后解决
