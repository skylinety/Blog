# SSD

## 寿命参数

### P/E

Program（加电、编程）、Erase（断电、擦除）

| type   | P/E 次     |
| ------ | ---------- |
| SLC    | 50000      |
| MLC    | 5000~10000 |
| 3D-TLC | 1000~3000  |
| 3D-QLC | 500~1000   |
| 3D-PLC | 100~500    |

### TBW

TB 写入（Terabytes Written）
TB 写入

### MTTF

平均无故障时间（Mean Time To Failures）
