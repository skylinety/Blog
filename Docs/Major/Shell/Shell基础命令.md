# Shell 基础命令

## ls

ls list，列出目录内容

| 参数 | 使用  | 描述                       |
| ---- | ----- | -------------------------- |
| 1    | ls -1 | 换行列出文件               |
| a    | ls -a | 列出所有文件，包括隐藏文件 |
| d    | ls -d | 列出目录本身，而不是其下内容 |


```shell
ls -d
```
列出的目录为当前目录，结果为.，要想列出当前目录下的所有目录使用
```shell
ls -d */
```
