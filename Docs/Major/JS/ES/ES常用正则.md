### 匹配定长字符

/\b.{m,n}\b/匹配指定长度的字符

```shell
 /\b.{2,5}\b/.test('aaaaaaa') // false
 /\b.{2,5}\b/.test('aa') // true
 /\b.{2,5}\b/.test('aaaaa') // true
```
\b 配置词边界
