# Python中的Math

## math内置
使用前引入math
```sh
import math
```

| 符号 | 含义    | 输入                         | 输出               | 含义                  |
| ---- | ------- | ---------------------------- | ------------------ | --------------------- |
| exp  | e的指数 | `math.exp(2)`                | `7.38905609893065` | $e^x$                 |
| log  | e的对数 | `math.log(7.38905609893065)` | `2.0`              | $\log_e{x}$或$\ln{x}$ |

## 全局方法

| 符号  | 含义   | 输入        | 输出  |
| ----- | ------ | ----------- | ----- |
| abs   | 绝对值 | `abs(-5)`   | `5`   |
| int   | 整型   | `int(1.11)` | `1`   |
| float | 浮点型 | `float(1)`  | `1.0` |
| max   | 最大值 | `max(1,2)`  | `2`   |


## 运算符

| 符号 | 含义 | 输入       | 输出   |
| ---- | ---- | ---------- | ------ |
| //   | 整除 | `5 // 2`   | `2`    |
| **   | 指数 | ` 2 ** 10` | `1024` |

## 其他

| 用法       | 输入   | 输出    |
| ---------- | ------ | ------- |
| 科学计数法 | `1e-3` | `0.001` |
