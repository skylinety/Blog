# 鹧鸪天·花重蓉城

---

&nbsp;
&nbsp;

花重蓉城三月天，倚栏红袖醉氲烟。
句芒折柳拂新靥，崔护携桃唤旧颜。

天晏晏，水涟涟，风轻丹落漫飞妍。
稚童不晓春光贵，尤与林鹃笑客眠。
